# Master Thesis EV Charging Simulation

This repository is used for my master thesis about smart electric vehicle charging.

The main part of the work is based on **EV2Gym**, a simulation environment for electric vehicle charging and vehicle-to-grid scenarios. I use EV2Gym to test charging strategies, inspect simulation outputs, and later compare simple uncontrolled charging with smarter charging methods.

## Current Work

- exploring how EV2Gym works,
- running example EV2Gym configurations,
- preparing a smaller thesis scenario with one EV, one charging station, PV production, and variable electricity prices,
- collecting rewards, power usage, and simulation statistics.
